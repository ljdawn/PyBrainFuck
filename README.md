Is a [brainfuck](http://zh.wikipedia.org/wiki/Brainfuck) interpreter written in Python.

  @ >+++++++++[<++++++++>-]<.>+++++++[<++++>-]<+.+++++++..+++.[-]>++++++++[<++++>-]<.>+++++++++++[<+++++>-]<.>++++++++[<+++>-]<.+++.------.--------.[-]>++++++++[<++++>-]<+.[-]++++++++++.

  $ ./PyBrainFuck.py example/HelloWorld1.bf
